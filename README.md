# 🎮 Reinforcement Learning Monorepo

## 📌 Overview

This repository is a **monorepo for Reinforcement Learning (RL) projects**. It is designed to manage multiple RL environments, agents, and experiments in a **clean, scalable, and modular structure**.

The repository focuses on building **intelligent agents** that learn optimal behavior through interaction with environments using modern RL techniques.

---

## 🎯 Goals

* Centralize all Reinforcement Learning projects in one place
* Enable reuse of **agents, environments, and training pipelines**
* Maintain a **clean and modular architecture**
* Support **fast experimentation and benchmarking**
* Build **scalable and research-oriented RL systems**

---

## 🚀 Tech Stack

### 🧑‍💻 Languages

* Python

### ⚙️ Frameworks & Libraries

* Stable-Baselines3 *(for RL algorithms)*
